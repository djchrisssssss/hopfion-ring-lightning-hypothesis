## 整體評價

假說的核心直覺——球狀閃電是環面而非球體——具有物理基礎且富有啟發性。文件格式、參考文獻品質、與先行研究的連結都做得不錯。但作為一份要被物理學界認真對待的假說，目前有幾個**嚴重的定量缺口**需要補上。

---

## P0 — 嚴重問題（必須解決，否則無法自洽）

### 問題 1：電阻性衰減時間尺度 — 致命矛盾

**這是整份假說最大的物理漏洞。**

在電阻性 MHD 中，磁螺旋度的衰減速率為：

```
dH_m/dt = -2η ∫ J·B d³r
```

磁擴散時間尺度為 τ_η ~ L²/η。大氣電漿（部分電離空氣）的磁擴散率 η 大約在 10²–10⁴ m²/s。即使取最樂觀的 η ~ 10² m²/s，對 L ~ 0.1 m 的結構：

```
τ_η ~ (0.1)² / 10² = 10⁻⁴ s = 0.1 ms
```

這是**微秒到毫秒等級**，不是秒。但假說聲稱壽命 5-10 秒。差距達 **4-5 個數量級**。

**假說必須解釋**：為什麼有效電阻率比普通大氣電漿低這麼多？不解決這個問題，「拓撲保護」就只是一句口號，而非定量機制。

**需要回答的具體問題**：什麼物理過程將 η 從 ~10² m²/s 降低到 ~10⁻³ m²/s 或更低？

---

### 問題 2：能量預算 — 數量級不足

用假說自己的參數估算儲存的磁能：

```
E_B = B²/(2μ₀) × V
```

取 B ~ 50 mT, R ~ 0.15 m（典型 25 cm 球狀閃電）：

```
E_B ≈ (50×10⁻³)² / (2 × 4π×10⁻⁷) × (4/3)π(0.15)³
    ≈ 1000 J/m³ × 0.014 m³
    ≈ 14 J
```

但觀測數據表顯示典型能量是 **8-80 kJ**。磁能比觀測能量低了 **3-4 個數量級**。

這意味著：
- 要嘛磁場遠比 50 mT 強（但那會產生其他可觀測效應）
- 要嘛能量不全是磁能（還有動能、熱能？比例是多少？）
- 要嘛觀測能量估計本身有問題

**假說需要完整的能量預算表**：E_total = E_magnetic + E_kinetic + E_thermal，各項均需定量估算。

---

### 問題 3：螺旋度注入來源 — 因果鏈斷裂

Taylor 弛豫 *守恆* 螺旋度，但不 *創造* 螺旋度。Ch. 6 描述的形成機制（步驟 1-5）沒有解釋：

**閃電放電如何注入非零的磁螺旋度？**

這不是顯而易見的。直線電流通道（典型雷擊）產生的是環形磁場，H_m = 0。要產生 H_m ≠ 0，需要：
- 電流路徑本身就有扭結（螺旋形閃電通道？）
- 多條電流通道互相纏繞
- 外部場（地磁場？）提供對稱性破缺

這是假說的**因果鏈斷裂點**：聲稱 Q_H ≠ 0 是穩定性的來源，但沒有解釋 Q_H 是怎麼從零變成非零的。

**需要回答的具體問題**：閃電放電過程中，什麼物理機制創造了淨磁螺旋度？

## P1 — 重要問題（應該補強）

### 問題 4：壽命公式是同義反覆

Ch. 8 的核心公式：

```
τ ∝ L / P_dissipation
```

這只是在說「壽命 = 能量儲量 / 耗散功率」——這是維度分析的恆等式，對任何系統都成立。它沒有告訴讀者 **為什麼 P_dissipation 足夠小**。

需要的是從第一原理推導的耗散功率表達式：

```
P_diss = η ∫ J² d³r + P_radiation + P_thermal_conduction
```

然後展示拓撲約束如何具體降低 J² 的空間積分（例如力自由場中 J × B = 0 意味著電流與磁場平行，減少焦耳加熱路徑）。

---

### 問題 5：光學假象論點缺乏定量支持

「球形是厚環面的投影」是本假說最原創的論點之一，但目前只是文字描述。需要：

- **合成影像模擬**：給定環面幾何（大半徑 R、小半徑 a、發光分佈），從不同觀測角度做光線追蹤，生成模擬影像
- 厚環面從側面看不會像球——會是橢圓或甜甜圈形。從什麼角度範圍看起來「像球」？統計上觀測者有多大機率從那個角度看？
- 與實際球狀閃電照片的比較

---

### 問題 6：缺少與 Spheromak 的比較

Spheromak 是實驗室中已經實現的環形電漿結構，具有連結的磁場線——本質上就是假說描述的東西。但實驗室 spheromak（有主動磁場容器）壽命也只有毫秒等級。

假說需要正面回應：**如果 spheromak 在實驗室都活不過毫秒，為什麼大氣中的類似結構能活幾秒？** 這直接回到問題 1（電阻率）。

參考文獻：Bellan, P.M. (2000). *Spheromaks*. World Scientific [38]。

## P2 — 中等問題（建議處理）

### 問題 7：4200 K 電漿的電離率問題

Cen 等人測量核心溫度 ~4200 K。用 Saha 方程估算，空氣在 4200 K 的電離分率極低（< 10⁻⁴）。這麼低的電離率能支撐假說全文假設的 MHD 描述嗎？

MHD 通常要求高度電離。可能的出路：
- 核心溫度實際更高，4200 K 是表面/平均值
- 即使弱電離，也可以用「部分電離 MHD」描述
- 但需要明確說明並引用相關理論
