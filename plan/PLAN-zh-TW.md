### 1.1 — 電阻性衰減時間尺度（問題 1）

**目標**：解釋 τ_η ~ 0.1 ms 與觀測 τ ~ 5-10 s 之間 4-5 個數量級的差距。

**研究方向**：

| 方案 | 機制 | 需要調查的關鍵文獻 |
|------|------|-------------------|
| A | **完全電離核心** — 若核心 T >> 4200 K（測量值為表面/平均值），Spitzer 電阻率 η_S ∝ T^(-3/2) 可將 η 降低數個數量級。在 T ~ 10⁵ K 時，η_S ~ 10⁻³ m²/s → τ_η ~ 10 s。 | Spitzer (1962) *Physics of Fully Ionized Gases*; Braginskii (1965) 傳輸係數 |
| B | **磁絕緣** — 拓撲結構本身抑制跨場輸運。電子被限制在磁力線上；徑向擴散需要跨場輸運，受 Bohm 擴散限制：D_⊥ ~ kT/(16eB)。 | Bohm 擴散; Chen (2016) *Introduction to Plasma Physics* |
| C | **湍流抑制** — 3D MHD 中磁螺旋度的逆級聯可將能量集中在大尺度，而耗散發生在小尺度，有效增加衰減時間。 | Frisch 等人 (1975); Biskamp (2003) *Magnetohydrodynamic Turbulence* |
| D | **雙區模型** — 高溫、完全電離的核心（低 η）被冷卻、弱電離的外殼（高 η 但充當絕緣體）包圍。拓撲結構僅存在於低 η 核心中。 | 類似於 tokamak 台座模型 |

**交付成果**：新章節「Ch. 8A — 電阻性壽命分析」，包含：
- 每個方案的 η 顯式計算
- 每個方案的 τ_η 結果
- 說明哪個方案最符合觀測
- 明確識別方案之間的可測試差異

**驗收標準**：至少一個物理上合理的方案在自洽參數下得出 τ_η ~ 1-10 s。

### 1.2 — 能量預算（問題 2）

**目標**：建構完整能量預算，展示 E_total ~ 8-80 kJ 是可實現的。

**分析計劃**：

```
E_total = E_magnetic + E_kinetic + E_thermal

E_magnetic = ∫ B²/(2μ₀) d³V
E_kinetic  = ½ ∫ ρv² d³V
E_thermal  = ∫ nkT/(γ-1) d³V
```

**逐步執行**：

1. **定義幾何**：大半徑 R、小半徑 a、縱橫比 A = R/a 的環面
2. **參數化場**：環面內的力自由 Beltrami 場 B(r)；指定 B₀（峰值場）
3. **計算 E_magnetic(B₀, R, a)**：在環面體積 V = 2π²Ra² 上積分
4. **計算 E_kinetic(ρ, v, R, a)**：電漿旋轉貢獻；使用理想氣體定律在測量 T 和大氣壓下的 ρ
5. **計算 E_thermal(T, n, R, a)**：使用 Cen 等人的 T ~ 4200 K（或雙區模型的較高核心 T）
6. **找到參數範圍**使 E_total ~ 8-80 kJ
7. **檢查自洽性**：所需的 B₀、v、T 值是否與其他約束矛盾？

**關鍵問題**：若 B ~ 50 mT 時 E_magnetic ~ 14 J，需要多大的 B 才能使 E_magnetic ~ 10 kJ？答案：B ~ 1.3 T。這物理上合理嗎？（可能太高——暗示能量以 E_kinetic 或 E_thermal 為主，而非 E_magnetic。）

**替代方案**：若能量主要是熱能（被磁拓撲困住的高溫電漿，~10⁴ K），那麼拓撲場充當**容器**而非**能量儲存庫**。這改變了敘事但可能更自洽。

**交付成果**：新章節「能量預算分析」，包含表格：

| 成分 | 公式 | 估算值 | 使用的參數 |
|------|------|--------|-----------|
| E_magnetic | ... | ... | ... |
| E_kinetic | ... | ... | ... |
| E_thermal | ... | ... | ... |
| **E_total** | — | **~XX kJ** | — |

**驗收標準**：E_total 在觀測範圍（8-80 kJ）的 3 倍因子內。

### 1.3 — 螺旋度注入機制（問題 3）

**目標**：識別在閃電放電過程中產生 H_m ≠ 0 的具體物理機制。

**研究方向**：

| 方案 | 機制 | 物理基礎 |
|------|------|---------|
| A | **地磁螺旋度注入** | 閃電電流 I 與環境地磁場 B_geo 交互作用。若 I 與 B_geo 不平行，互螺旋度 H_m = ∫ A_geo · B_lightning d³r ≠ 0。地磁場打破了閃電通道的方位角對稱性。 |
| B | **螺旋形回擊** | 高速攝影顯示閃電通道並非直線而是蜿蜒路徑。螺旋形電流路徑本身具有自螺旋度。若螺旋角為 θ，則 H_m ∝ sin(2θ)。 |
| C | **多通道重聯** | 當一次閃電放電的多個回擊或分支重聯時，電流路徑的互相纏繞產生互螺旋度。這類似於太陽耀斑通過磁通湧現注入螺旋度。 |
| D | **湍流螺旋度級聯** | 雷暴環境中的大氣湍流提供隨機螺旋度庫。磁重聯選擇性地捕獲單一符號的螺旋度（類似於發電機理論中的 α 效應）。 |

**最有前途的方案**：方案 A（地磁注入）— 可量化、始終存在，且與閃電電流和 B_geo 之間的夾角正弦成正比，這可以解釋球狀閃電頻率的地理變化。

**交付成果**：新章節「Ch. 6A — 螺旋度注入機制」，包含：
- 單次回擊注入的 H_m 定量估算
- 與地磁傾角的依賴關係
- 與觀測到的球狀閃電參數所需 H_m 的比較

**驗收標準**：估算的 H_m 注入量足以在典型參數下產生 Q_H ≥ 1。

## 第二階段：解決 P1 重要問題

### 2.1 — 第一原理壽命推導（問題 4）

**目標**：用推導出的表達式取代 τ ∝ L/P_diss。

**方法**：
1. 從力自由 Beltrami 場出發：∇ × B = αB
2. Taylor 弛豫：系統在恆定螺旋度下演化至最低能量
3. 在電阻性 MHD 中，螺旋度比能量衰減慢（Berger 1984）：dE/dt ∝ α²η 而 dH/dt ∝ αη
4. 為 Hopfion 幾何顯式推導 τ_helicity vs τ_energy
5. 展示 τ_helicity / τ_energy = (αL)² >> 1（對適當的 α）

**交付成果**：更新 Ch. 8，包含推導展示：
```
τ_helicity = L² / η_eff
τ_energy = 1/(α²η_eff)
```
其中 α ~ 1/a（小半徑的倒數），展示螺旋度比能量長壽因子 (αa)² ~ (L/a)²。

### 2.2 — 光學假象合成影像（問題 5）

**目標**：生成發光厚環面從多個視角觀看的合成影像。

**方法**：
1. 定義環面幾何：R（大半徑）、a（小半徑），a/R ~ 0.5-0.8（厚環面）
2. 定義發射率分佈：j(r) ∝ exp(-((r-R)² + z²)/(2σ²))（柱坐標）
3. 沿視線積分，觀測角 θ = 0°（正面）、30°、45°、60°、90°（側面）
4. 加入大氣散射（高斯模糊 σ_scatter ~ 0.1R）
5. 生成影像並與球狀閃電照片比較

**工具**：Python + matplotlib（2D 柱密度積分）或簡單光線追蹤腳本。

**交付成果**：新附錄「發光環面合成影像」，包含：
- 5-6 張不同視角的合成影像

**驗收標準**：產生球形外觀的視角 ≥ 60% 的立體角（與 ~80% 報告描述「球形」一致）。

### 2.3 — Spheromak 比較（問題 6）

**目標**：明確比較假說與實驗室 spheromak 並解釋差異。

**關鍵比較**：

| 性質 | 實驗室 Spheromak | Hopfion 環閃電（假說） |
|------|------------------|----------------------|
| 尺寸 | ~0.1-1 m | ~0.1-0.3 m |
| 壽命 | ~0.1-1 ms | ~5-10 s |
| 約束 | 外部磁通守恆器（金屬壁） | 自約束（無壁） |
| 溫度 | ~10-100 eV (~10⁵ K) | ~0.4 eV (~4200 K 表面) |
| 密度 | ~10¹⁹-10²⁰ m⁻³ | ~10²⁴-10²⁵ m⁻³（大氣） |
| 磁場 | ~0.1-1 T | ~10-100 mT（預測） |
| β（電漿/磁壓比） | ~0.01-0.1 | ~1-10（高 β） |

**需要論證的關鍵差異**：實驗室 spheromak 是**低 β、真空邊界**系統，壁提供磁通守恆邊界。大氣 hopfion 閃電是**高 β** 系統，**高密度中性氣體提供慣性約束** — 沉重的中性大氣充當「壁」。這從根本上改變了衰減動力學。

**交付成果**：新章節「與實驗室 Spheromak 的比較」，包含上表及壽命差異的物理論證。

**目標**：論證 MHD 在觀測溫度下的適用性。

**方法**：

4. 論證即使在 4200 K，離子-中性粒子耦合對此處相關的低頻現象（ω << ω_ci）也是足夠的
